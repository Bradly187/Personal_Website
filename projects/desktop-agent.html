Personal Desktop Agent

Multimodal accessibility agent enabling hands-free Windows control through voice, gaze, gesture, tilt, and touch — powered by local GPU inference with AWS cloud fallback. Built to solve a real problem: computing access for people who can't use a keyboard.

// Architecture
MCP Server
Exposes 14 desktop control tools (mouse, keyboard, screen, windows) to Claude via stdio transport.
iPad Bridge
WebSocket server handling 13 incoming sensor message types and 4 outgoing reply types, dispatching to processing pipelines.
Fusion Engine
Merges 10 input modalities at 60Hz using priority-based routing with confidence gating.
Hybrid Coordinator
5-gate routing engine (Gate 0 privacy + Gates 1–4) deciding local vs. cloud inference based on confidence, complexity, and GPU load.
Local Inference
Whisper large-v3 + Llama 3.1 running on RTX 5090 for sub-600ms voice command processing.
Continuous Trainer
Background learning system that adapts routing thresholds and gesture confidence floors; delegates storage to AgentDB.
Gesture + LiDAR
MediaPipe hand landmark detection for POINT/PINCH/OPEN_PALM/FIST; LiDAR depth map (256×192) used for 3D pinch distance validation at <30mm threshold.
Domain Classifier
Keyword-scoring classifier routes commands to the accessibility pipeline or one of 5 dev domains (CODE/MATH/VISION/PLAN/GENERAL).
Dev Agent
Plan→execute→reflect agentic loop; selects specialist model per domain via ModelRouter; emits 5 dev verbs (WRITE_FILE, RUN_TERMINAL, EXPLAIN, SEARCH_WEB, READ_SCREEN).
AgentDB / AnalyticsDB
aiosqlite 12-table pipeline store + DuckDB benchmark analytics; MiniLM semantic few-shot retrieval for routing decisions.

// Routing gates
Privacy Gate (0)Detects sensitive patterns (passwords, tokens, SSNs, card numbers) — forces local silently, no cloud exposure.
Confidence Gate (1)Whisper logprob ≥ -1.0 or gesture confidence ≥ 0.60; low-confidence voice triggers re-transcription.
Complexity Gate (2)Token count proxy ≤ 40 words; complex/multi-step queries routed to specialist models via DevAgent.
VRAM Gate (3)≥ 8 GB free VRAM required (absolute floor) — tuned for RTX 5090 running alongside Whisper.
Latency Budget Gate (4)Estimated local inference latency must be within the latency target for that input modality.
