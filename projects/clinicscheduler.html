// Features
Role-Based Access Control Distinct permission levels for admin, clinician, and patient roles throughout the application.
Patient Management Full CRUD for patient records including demographics, contact info, and medical history.
Appointment Scheduling Calendar-based booking with conflict detection, reminders, and status tracking.
HL7 FHIR Integration Structured health data exchange using the FHIR R4 standard for interoperability.
Identity Pipeline Integration Receives verified patient and therapist records from the AWS serverless onboarding pipeline.
Cross-Platform Client .NET MAUI mobile client alongside a Blazor web interface — one codebase, multiple targets.
